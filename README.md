# RS040-escape-from-college
Escape From College

Escape form College is 2D game. You play as student, who need to pick up all exam papers.
Hurry up before professors catch you!


Requirements - QT


CONTROLS:
WASD - to move
SPACE - to perfome action
ESC - close game
